# Política de Privacidade - ERP Checkbox Marker

## Extensão: ERP Checkbox Marker - SaaS

**Última atualização:** 09/03/2026

---

### 1. Coleta de Dados

Esta extensão **não coleta** dados pessoais dos usuários. 

Os únicos dados armazenados localmente no navegador são:
- **Email de validação**: Utilizado apenas para verificar autorização de acesso
- **Presets de lojas**: Criados pelo próprio usuário para uso pessoal

---

### 2. Uso dos Dados

Os dados armazenados localmente são utilizados exclusivamente para:
- Validar o acesso do usuário à extensão
- Permitir o uso dos recursos de marcação de checkboxes no ERP
- Armazenar presets de lojas criados pelo usuário

---

### 3. Compartilhamento

**Nenhum dado é compartilhado com terceiros.**

A extensão faz apenas duas requisições externas:
- **ajudanet.github.io**: Para validação de email autorizado
- **erp.bluesoft.com.br**: Para funcionalidade principal no ERP

---

### 4. Armazenamento

Todos os dados são armazenados exclusivamente no dispositivo do usuário através da API de armazenamento local do Chrome (chrome.storage.local).

---

### 5. Cookies

Esta extensão **não utiliza cookies**.

---

### 6. Contato

Para dúvidas sobre esta política de privacidade, entre em contato com o desenvolvedor.
